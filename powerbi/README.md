# PowerBI & Presentation Outputs

This directory acts as the collection bin for all visualization and reporting artifacts generated by the Consumer360 analytical engine. 

## Workflow

1. Data enriched from the Python ETL scripts is dumped as an aggregated Excel object (`Consumer360_PowerBI_Data.xlsx`) which serves as structurally optimized input for PowerBI Desktop `.pbix` visualization.
2. The core python framework generates standalone reporting deliverables in standard HTML (`Consumer360_Dashboard.html`) using Plotly.
3. Automatically generated Churn Risk intervention reports (`Consumer360_Executive_Deck.pptx`) are also outputted here using Python-PPTX.

All files enclosed within this target are typically refreshed whenever the core pipeline (`main_pipeline.py`) is successfully completed.
